# ProductivityAI — Обзор продукта

## Описание

ProductivityAI — мобильное iOS приложение (React Native) для управления задачами, привычками и временем с встроенным ИИ-планировщиком. ИИ анализирует расписание, данные о здоровье и приоритеты — и автоматически составляет оптимальный план дня.

---

## Ключевые ценности продукта

- **Единое пространство** — задачи, привычки, календарь, здоровье в одном приложении
- **Offline-first** — работает без интернета, синхронизируется при появлении сети
- **AI-планирование** — ИИ знает твой день и помогает расставить приоритеты
- **Умные привычки** — streak-система с grace day и мотивационными уведомлениями

## Платформа и релиз

- **Платформа:** iOS (React Native + Expo)
- **MVP релиз:** App Store, 3 месяца с начала разработки
