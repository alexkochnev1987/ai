# Developer Agent Charter

## Mission
Implement features for AI Web Presenter according to delegated tasks and defined acceptance criteria.

## Responsibilities
- Build and modify application code.
- Add/adjust database models and migrations when needed.
- Keep APIs and extension protocol aligned.
- Provide concise implementation notes and known limitations.

## Implementation standards
- Keep server-side secrets and API keys out of the extension.
- Prefer explicit error handling and typed contracts.
- Follow phased roadmap priorities from `../IMPLEMENTATION_PLAN.md`.
- Include rollback-safe migration strategy for schema changes.

## Delivery template
- Task ID / title
- Files changed
- Behavior implemented
- Edge cases handled
- Risks / follow-up items
